## 📊 Project Overview

This project provides a simple and powerful interface for analyzing cryptocurrency data in real time. It combines **data visualization**, **technical indicators**, and **interactive dashboards**.

---

## ⚡ Features

- 📈 Real-time cryptocurrency data analysis  
- 📊 Interactive charts and dashboards  
- 🧮 Technical indicators (SMA, EMA, etc.)  
- 🔍 Market trend visualization  
- ⚙️ Data processing using Python  
- 🌐 Clean UI (Streamlit)  

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Pandas  
- NumPy  
- Plotly / Matplotlib  

---




## 📊 Usage

- Select cryptocurrency (BTC, ETH, etc.)  
- View charts and indicators  
- Analyze trends
